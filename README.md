# emmanuelchucks.com

My personal portfolio website to showcase my latest work.

## Features

- Contact form with SendGrid
- Fully responsive
- Dark mode
- Accessibility
- Lighthouse score

## Running Locally

```bash
git clone https://github.com/emmanuelchucks/emmanuelchucks.com.git
cd emmanuelchucks.com
npm i
npm run dev
```

## Environment Variables

SENDGRID_API_KEY=

## Built Using

- [Next.js](https://nextjs.org/)
- [Tailwind](https://tailwindcss.com/)
- [Vercel](https://vercel.com) (Deployment)
